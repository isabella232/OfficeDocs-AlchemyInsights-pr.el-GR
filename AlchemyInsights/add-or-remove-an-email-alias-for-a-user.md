# <a name="add-or-remove-an-email-address-for-a-user"></a>Προσθήκη ή κατάργηση διεύθυνσης ηλεκτρονικού ταχυδρομείου για ένα χρήστη

Μπορείτε να έχετε περισσότερες από μία διευθύνσεις ηλεκτρονικού ταχυδρομείου για ένα χρήστη, αλλά τα πρόσθετα  *ψευδώνυμα ηλεκτρονικού ταχυδρομείου*  μπορούν να χρησιμοποιηθούν μόνο για τη λήψη μηνυμάτων ηλεκτρονικού ταχυδρομείου. Εάν θέλετε να στείλετε μήνυμα ηλεκτρονικού ταχυδρομείου με ψευδώνυμο, δημιουργήστε ένα [κοινόχρηστο γραμματοκιβώτιο](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox). Με αυτόν τον τρόπο, θα λάβετε το ψευδώνυμο για την αποστολή και λήψη αλληλογραφίας.

# <a name="how-to-add-and-manage-admins"></a>Τρόπος προσθήκης και διαχείρισης διαχειριστών

Με βάση την περιγραφή του προβλήματος, έχουμε βρει μια λύση για εσάς. Οι περισσότεροι πελάτες είχαν τη δυνατότητα να επιλύσουν το πρόβλημά τους μόνοι τους μετά την τεκμηρίωση.

Για να διαχειριστείτε το λογαριασμό χρέωσής σας για μια Σύμβαση πελάτη της Microsoft (MCA), μπορείτε να χρησιμοποιήσετε διαφορετικούς ρόλους με το επιθυμητό επίπεδο πρόσβασης. Αυτοί οι ρόλοι είναι εκτός από τους ενσωματωμένους ρόλους υπηρεσίας Azure που σας βοηθούν να ελέγχετε τους πόρους σας.

**Για να προσθέσετε ρόλους χρέωσης στην πύλη Azure:**

1. Πραγματοποιήστε είσοδο στην [πύλη Azure.](https://portal.azure.com/)
2. Αναζήτηση για *Διαχείριση κόστους + Χρεώσεις*.
3. Επιλέξτε το στοιχείο ελέγχου της Access (IAM) σε ένα εύρος όπως ο λογαριασμός χρέωσης, το προφίλ χρέωσης ή η ενότητα τιμολογίου όπου θέλετε να δώσετε πρόσβαση.
4. Η σελίδα στοιχείου ελέγχου της Access (IAM) παραθέτει τους χρήστες και τις ομάδες που έχουν εκχωρηθεί σε κάθε ρόλο για αυτό το εύρος.
5. Για να δώσετε πρόσβαση σε ένα χρήστη, **επιλέξτε "Προσθήκη"** από το επάνω μέρος της σελίδας. Στην αναπτυσσόμενη *λίστα* "Ρόλος", επιλέξτε ένα ρόλο. Εισαγάγετε τη διεύθυνση ηλεκτρονικού ταχυδρομείου του χρήστη στον οποίο θέλετε να δώσετε πρόσβαση. Επιλέξτε **"Αποθήκευση"** για να αναθέσετε το ρόλο.
6. Για να καταργήσετε την πρόσβαση για ένα χρήστη, επιλέξτε το χρήστη με την ανάθεση ρόλου που θέλετε να καταργήσετε. Επιλέξτε **"Κατάργηση".**

# <a name="restore-a-deleted-site"></a>Επαναφορά διαγραμμένης τοποθεσίας

Όταν ένας διαχειριστής διαγράφει μια τοποθεσία SharePoint, τοποθετείται στον Κάδο Ανακύκλωσης συλλογής τοποθεσιών, όπου διατηρείται για 93 ημέρες πριν διαγραφεί οριστικά. Για να επαναφέρετε την τοποθεσία:
  
1. Στο νέο κέντρο διαχείρισης SharePoint, κάντε κλικ στην **επιλογή "Κάδος Ανακύκλωσης"** στην κορδέλα. 
    
2. Επιλέξτε το πλαίσιο ελέγχου δίπλα στη συλλογή τοποθεσιών που θέλετε να επαναφέρετε.
    
3. Κάντε κλικ **στην επιλογή "Επαναφορά διαγραμμένων στοιχείων".**
    
Για να επαναφέρετε μια διαγραμμένη τοποθεσία επικοινωνίας, μπορείτε να χρησιμοποιήσετε το νέο SharePoint διαχείρισης. Διαφορετικά, πρέπει να χρησιμοποιήσετε το Microsoft PowerShell. Για να επαναφέρετε μια τοποθεσία που ανήκει σε μια Microsoft 365, πρέπει να επαναφέρετε την ομάδα στο Exchange διαχείρισης. Οι ομάδες μπορούν να αποκατασταθούν για 30 ημέρες μετά τη διαγραφή τους.

# <a name="sync-contacts"></a>Συγχρονισμός επαφών

Outlook συγχρονίζει τις επαφές που δημιουργείτε στα [Άτομα](https://outlook.live.com/people/) σε άλλες συσκευές και εφαρμογές, όπως Outlook για Android και Outlook για iOS. Μάθετε πώς μπορείτε να [δημιουργήσετε επαφές.](https://support.office.com/article/5b909158-036e-4820-92f7-2a27f57b9f01)

Οι επαφές που έχουν εισαχθεί (για παράδειγμα, οι επαφές από άλλο λογαριασμό ηλεκτρονικού ταχυδρομείου που έχετε συνδέσει στο λογαριασμό Outlook.com) δεν συγχρονίζονται. Μάθετε πώς μπορείτε να [εισαγάγετε επαφές.](https://support.office.com/article/285a3b55-8d93-4ac8-93df-43fffd13b2f1)

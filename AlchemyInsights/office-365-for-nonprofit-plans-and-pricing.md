# <a name="microsoft-365-for-nonprofit-plans-and-pricing"></a>Microsoft 365 προγραμμάτων και τιμολόγησης για Μη Κερδοσκοπικούς Οργανισμούς

Οι μη κερδοσκοπικοί οργανισμοί που πληρούν τις προϋποθέσεις μπορούν να λάβουν Microsoft 365 μη κερδοσκοπικούς οργανισμούς ως δωρεά ή να κάνουν αναβάθμιση σε σύνθετες δυνατότητες με σημαντική έκπτωση. Για να πληροίτε τις προϋποθέσεις για ένα Microsoft 365 πρόγραμμα για Μη Κερδοσκοπικούς Οργανισμούς, πρέπει να έχετε [αναγνωρίσει τη φιλανθρωπική σας κατάσταση](https://go.microsoft.com/fwlink/p/?LinkID=330253) στη χώρα σας και να υπογράψετε μια πολιτική που δεν αφορά τη Microsoft. Η Microsoft διατηρεί το δικαίωμα να ελέγξει την καταλληλότητα οποιαδήποτε στιγμή και να αναστείλει την υπηρεσία για οργανισμούς που δεν πληροί τις προϋποθέσεις.
  
Για περισσότερες πληροφορίες, ανατρέξτε στις [Συνήθεις ερωτήσεις.](https://products.office.com/nonprofit/office-365-nonprofit)
  
- [Εξετάστε τα προγράμματα για Μη Κερδοσκοπικούς Οργανισμούς](https://products.office.com/nonprofit/office-365-nonprofit-plans-and-pricing?tab=1)

- [Σύγκριση όλων των συνδρομών Microsoft 365](https://products.office.com/business/compare-more-office-365-for-business-plans)

# <a name="power-bi-license-management"></a>Διαχείριση αδειών χρήσης Power BI

Η διαχείριση αδειών χρήσης του Power BI μπορεί να ολοκληρωθεί στο Κέντρο διαχείρισης Microsoft 365.

- **Για να αγοράσετε το Power BI:** Μεταβείτε στις **υπηρεσίες** \> **[αγορών χρέωσης.](https://go.microsoft.com/fwlink/p/?linkid=868433)** Χρησιμοποιήστε το πλαίσιο αναζήτησης για να αναζητήσετε το "Power BI" και, στη συνέχεια, επιλέξτε το πρόγραμμα και τον αριθμό των χρηστών που θέλετε να αγοράσετε. Για περισσότερες πληροφορίες, ανατρέξτε στο [θέμα "Αγορά διαφορετικής συνδρομής".](https://docs.microsoft.com/microsoft-365/commerce/try-or-buy-microsoft-365#buy-a-different-subscription)
- **Για να εκχωρήσετε άδειες χρήσης του Power BI:** Μεταβείτε στην ενότητα **"Άδειες**  >  **[χρήσης χρέωσης".](https://go.microsoft.com/fwlink/p/?linkid=842264)** Για περισσότερες πληροφορίες, ανατρέξτε στο [θέμα "Εκχώρηση αδειών χρήσης σε χρήστες".](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users)
- Για να προσθέσετε ή να καταργήσετε άδειες χρήσης για το **Power BI:** Μεταβείτε στην επιλογή "Χρέωση των προϊόντων σας", βρείτε τη συνδρομή που θέλετε να αλλάξετε, επιλέξτε "Περισσότερες επιλογές" (τρεις τελείες) και, στη συνέχεια, επιλέξτε "Αγορά αδειών   >  **[](https://go.microsoft.com/fwlink/p/?linkid=842054)** **χρήσης".**  Για να καταργήσετε άδειες χρήσης, στο επάνω μέρος του παραθύρου **"Αγορά αδειών χρήσης",** επιλέξτε **"Κατάργηση αδειών χρήσης".** Για περισσότερες πληροφορίες, ανατρέξτε στο [θέμα "Αγορά ή κατάργηση αδειών χρήσης".\](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses)
**ΠΑΡΑΚΟΛΟΥΘΉΣτε ένα βίντεο** που δείχνει πώς μπορείτε να το κάνετε αυτό: [Αγορά νέων αδειών χρήσης](https://go.microsoft.com/fwlink/p/?linkid=2154857)

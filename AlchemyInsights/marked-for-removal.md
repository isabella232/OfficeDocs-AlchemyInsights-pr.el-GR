# <a name="marked-for-removal"></a>Επισημάνθηκε για κατάργηση

Ο μισθωτής σας μπορεί να έχει επισημανθεί για κατάργηση, επειδή δεν έχουν αγοραστεί και εκχωρηθεί άδειες χρήσης ή ο μισθωτής έχει λήξει. 

Εάν έχει λήξει, μπορείτε να ενεργοποιήσετε εκ νέου το μισθωτή σας. Ανατρέξτε στο θέμα [Επανενεργοποίηση της συνδρομής σας](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/reactivate-your-subscription?view=o365-worldwide) για περισσότερες πληροφορίες.

Για να εκχωρήσετε άδειες χρήσης σε χρήστες, ανατρέξτε στο θέμα [Προσθήκη χρηστών μεμονωμένα ή μαζικά](https://support.office.com/article/Assign-or-remove-licenses-for-Office-365-for-business-997596b5-4173-4627-b915-36abac6786dc).

# <a name="youre-missing-out-error-message"></a>Μήνυμα σφάλματος «Λείπει»

Αν έχετε έναν μισθωτή για εκπαιδευτικά ιδρύματα, πρέπει να ενεργοποιήσετε το Microsoft Teams με μη αυτόματο τρόπο. Για να το κάνετε αυτό, μεταβείτε στην επιλογή του κέντρου διαχείρισης του Microsoft 365 **Ρυθμίσεις > Ρυθμίσεις > Microsoft Teams**. Εάν το Microsoft Teams δεν υπάρχει στη λίστα υπηρεσιών, απενεργοποιήστε την επιλογή **Δοκιμάστε το νέο κέντρο διαχείρισης** για να βρείτε τις **Ρυθμίσεις > Υπηρεσίες και πρόσθετα > Microsoft Teams**. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Ενεργοποίηση του Microsoft Teams για το σχολείο σας](https://docs.microsoft.com/microsoft-365/education/intune-edu-trial/enable-microsoft-teams#enable-microsoft-teams-for-your-school-1).

**Σημαντικό**: Ίσως χρειαστεί να περιμένετε έως και 24 ώρες για να τεθούν σε ισχύ οι αλλαγές.

# <a name="verify-your-yammer-tenant-is-in-native-mode"></a>Επαλήθευση ότι ο Yammer σας βρίσκεται σε εγγενή λειτουργία

Αυτό το βήμα είναι προαιρετικό. Στην εγγενή λειτουργία, όλοι Yammer χρήστες βρίσκονται στο Azure Active Directory (Azure AD), όλες οι ομάδες είναι Microsoft 365 Ομάδες και όλα τα αρχεία αποθηκεύονται στο SharePoint Online. Ο Yammer μισθωτή σας πρέπει να βρίσκεται στην εγγενή λειτουργία για να επιτρέπει στις πολιτικές συμμόρφωσης επικοινωνίας να σαρώνουν και να εντοπίζουν συνομιλίες με κινδύνους σε ιδιωτικά μηνύματα και συνομιλίες κοινότητας σε Yammer.  
  
Για περισσότερες πληροφορίες σχετικά με τη ρύθμιση Yammer σε εγγενή λειτουργία, ανατρέξτε στο θέμα Επισκόπηση της [εγγενούς](https://go.microsoft.com/fwlink/?linkid=2129829) λειτουργίας Yammer στο Microsoft 365 και ρύθμιση παραμέτρων του δικτύου Yammer για εγγενή λειτουργία [για Microsoft 365.](https://go.microsoft.com/fwlink/?linkid=2129772)

# <a name="use-jamf-to-configure-microsoft-edge-policy-settings-on-macos"></a>Χρησιμοποιήστε το Jamf για να ρυθμίσετε τις Microsoft Edge πολιτικής στο macOS

Για να Microsoft Edge (Σταθερό κανάλι 81 ή νεότερη έκδοση) σε macOS, χρησιμοποιήστε ένα αρχείο διακήρυξης πολιτικής Microsoft Edge στο Jamf Pro 10.19.

Για περισσότερες πληροφορίες, ανατρέξτε [στο θέμα Microsoft Edge ρυθμίσεις πολιτικής στο macOS με Το Jamf.](https://go.microsoft.com/fwlink/?linkid=2134761)

# <a name="issues-with-credentials"></a>Προβλήματα με τα διαπιστευτήρια

Πλατφόρμα ταυτοτήτων της Microsoft και η ροή διαπιστευτηρίων προγράμματος-πελάτη [OAuth 2.0](https://docs.microsoft.com/azure/active-directory/develop/v2-oauth2-client-creds-grant-flow) περιγράφει τον τρόπο με τον οποίο μπορείτε να προγραμματίσετε απευθείας σε σχέση με τη ροή εκχώρησης διαπιστευτηρίων προγράμματος-πελάτη OAuth 2.0.

**Πώς μπορώ να διαχειριστώ τον κωδικό πρόσβασης ή τα διαπιστευτήρια πιστοποιητικού μιας εφαρμογής;**

Στο Azure CLI, μπορείτε να χρησιμοποιήσετε τα διαπιστευτήρια της εφαρμογής [az ad app](https://docs.microsoft.com/cli/azure/ad/app/credential) για να διαγράψετε, να δημιουργήσετε μια λίστα ή να επαναφέρετε τον κωδικό πρόσβασης ή τα διαπιστευτήρια πιστοποιητικού μιας εφαρμογής.

**Πώς οι χρήστες μου επαναφέρουν τους κωδικούς πρόσβασής τους;**

Οι χρήστες πρέπει να [εγγραφούν για επαναφορά κωδικού πρόσβασης από το χρήστη για](https://docs.microsoft.com/azure/active-directory/user-help/active-directory-passwords-reset-register) να μπορούν να επαναφέρουν τους κωδικούς πρόσβασής τους. Μετά την εγγραφή ενός χρήστη, μπορεί να ακολουθήσει τις οδηγίες σε αυτό το άρθρο για να επαναφέρει τον κωδικό πρόσβασής του: [Επαναφέρετε τον κωδικό πρόσβασης της εργασίας ή του σχολείου σας.](https://docs.microsoft.com/azure/active-directory/user-help/user-help-reset-password#how-to-reset-or-unlock-your-password-for-a-work-or-school-account)

**Πώς αλλάζουν οι χρήστες τους κωδικούς πρόσβασής τους;**

Οι χρήστες μπορούν να ακολουθήσουν τα βήματα σε αυτό το άρθρο για να αλλάξουν τους κωδικούς πρόσβασής τους: [Πώς μπορείτε να αλλάξετε τον κωδικό πρόσβασής σας.](https://docs.microsoft.com/azure/active-directory/user-help/user-help-reset-password#how-to-change-your-password)
Μπορούν επίσης να [διαχειρίζονται κωδικούς πρόσβασης εφαρμογών για επαλήθευση σε δύο βήματα.](https://docs.microsoft.com/azure/active-directory/user-help/multi-factor-authentication-end-user-app-passwords)

**Ο χρήστης εμφανίζεται σφάλμα κατά την αλλαγή ή την επαναφορά του κωδικού πρόσβασής του**

Αυτή η σύνδεση θα παρέχει πληροφορίες σχετικά με συνήθη προβλήματα που μπορεί να προκύψουν όταν ένας χρήστης προσπαθεί να επαναφέρει τον κωδικό πρόσβασής του: [Συνήθη προβλήματα και λύσεις](https://docs.microsoft.com/azure/active-directory/user-help/user-help-reset-password#common-problems-and-their-solutions)

**Έχω πρόβλημα με την επαναφορά του κωδικού πρόσβασης ενός χρήστη**

- Βεβαιωθείτε ότι έχετε εξουσιοδότηση για την επαναφορά κωδικών πρόσβασης. *Μόνο οι καθολικοί χρήστες, οι κωδικοί πρόσβασης και οι διαχειριστές χρηστών μπορούν να επαναφέρουν κωδικούς πρόσβασης χρηστών.* Οι καθολικοί διαχειριστές μπορούν επίσης να επαναφέρουν τους κωδικούς πρόσβασης άλλου διαχειριστή με δικαιώματα.

- Βεβαιωθείτε ότι έχετε κατανοήσει τις απαιτήσεις άδειας χρήσης:

  - Πρέπει να σας έχει εκχωρηθεί τουλάχιστον μία άδεια χρήσης στον οργανισμό σας:
    - **Χρήστες μόνο στο cloud** - Οποιαδήποτε Office 365 (O365) επί πληρωμή SKU ή Azure AD Basic
    - **Χρήστες cloud ή/και** εσωτερικής εγκατάστασης - Azure AD Premium P1 ή P2, Φορητότητα για μεγάλες επιχειρήσεις + Ασφάλεια (EMS) ή Secure Productive Enterprise (SPE)
    - Για να μάθετε περισσότερα σχετικά με τις απαιτήσεις παραχώρησης αδειών χρήσης, ανατρέξτε στο θέμα [Απαιτήσεις παραχώρησης αδειών χρήσης](https://docs.microsoft.com/azure/active-directory/active-directory-passwords-licensing)για επαναφορά κωδικού πρόσβασης από το Azure AD από το χρήστη.
- Για να επαναφέρετε τον κωδικό πρόσβασης ενός χρήστη, βρείτε το χρήστη στο Azure AD. Στη συνέχεια, στη λάμα επισκόπησης για αυτόν το χρήστη, κάντε κλικ στο κουμπί "επαναφορά κωδικού πρόσβασης".

**Το κουμπί επαναφοράς κωδικού πρόσβασης είναι γκρι**

Δεν έχετε εξουσιοδότηση να επαναφέρετε τους **κωδικούς** πρόσβασης αυτού του χρήστη. *Μόνο οι καθολικοί χρήστες, οι κωδικοί πρόσβασης και οι διαχειριστές χρηστών μπορούν να επαναφέρουν κωδικούς πρόσβασης χρηστών.* Οι καθολικοί διαχειριστές μπορούν επίσης να επαναφέρουν τους κωδικούς πρόσβασης άλλου διαχειριστή με δικαιώματα.

**Δεν βλέπω τη λάμα επαναφοράς κωδικού πρόσβασης**

Δεν έχετε εξουσιοδότηση για την επαναφορά κωδικών πρόσβασης. *Μόνο οι καθολικοί χρήστες, οι κωδικοί πρόσβασης και οι διαχειριστές χρηστών μπορούν να επαναφέρουν κωδικούς πρόσβασης χρηστών.* Οι καθολικοί διαχειριστές μπορούν επίσης να επαναφέρουν τους κωδικούς πρόσβασης άλλου διαχειριστή με δικαιώματα.

**Δεν βλέπω τη blade ενοποίησης εσωτερικής εγκατάστασης στην επαναφορά κωδικού πρόσβασης**

- Η blade ενοποίησης εσωτερικής εγκατάστασης εμφανίζεται μόνο σε υβριδικά περιβάλλοντα , που σημαίνει ότι χρησιμοποιείτε την επαναφορά κωδικού πρόσβασης για το χειρισμό των κωδικών πρόσβασης των χρηστών εσωτερικής εγκατάστασης.

- Δεν βλέπετε αυτή τη λάμα εάν:

  - Δεν χρησιμοποιείτε τη δυνατότητα εγγραφής κωδικού πρόσβασης
  - Υπάρχει πρόβλημα με την εγκατάσταση/συνδεσιμότητα της εγγραφής κωδικού πρόσβασης
  - Υπάρχει πρόβλημα με την εγκατάσταση/συνδεσιμότητα του Azure AD Σύνδεση
  - Για περισσότερα βήματα αντιμετώπισης προβλημάτων για προβλήματα με την επαναφορά κωδικού πρόσβασης, ανατρέξτε στο θέμα Αντιμετώπιση [προβλημάτων με την επαναφορά κωδικού πρόσβασης](https://docs.microsoft.com/azure/active-directory/authentication/troubleshoot-sspr-writeback)

**Δεν ξέρω πώς μπορείτε να επαναφέρετε τον κωδικό πρόσβασης ενός χρήστη**

1. Πραγματοποιήστε είσοδο στην πύλη Azure ως κατάλληλος διαχειριστής.
2. Μεταβείτε στη σελίδα **"Χρήστες και ομάδες",** επιλέξτε **"Όλοι οι χρήστες".**
3. Επιλέξτε ένα χρήστη από τη λίστα.
4. Για τον επιλεγμένο χρήστη, επιλέξτε **"Επισκόπηση" και,** στη συνέχεια, στη γραμμή εντολών, επιλέξτε **"Επαναφορά κωδικού πρόσβασης".**
5. Επιλέξτε το **κουμπί "Επαναφορά κωδικού** πρόσβασης" και ακολουθήστε τις οδηγίες στην οθόνη.
    - Μόνο οι επαναφέρεις που εκτελούνται μέσω της επαναφοράς κωδικού πρόσβασης υποστήριξης της πύλης **Azure.**

**Επαναφέρω τον κωδικό πρόσβασης ενός χρήστη εσωτερικής εγκατάστασης από την πύλη Διαχείριση Office 365 ή την εφαρμογή Office 365 για κινητές συσκευές, αλλά ο χρήστης εξακολουθεί να μην μπορεί να κάνει είσοδο**

Η εγγραφή κωδικού πρόσβασης δεν υποστηρίζεται σε αυτήν την πύλη. Επαναφέρετε τον κωδικό πρόσβασης του χρήστη ξανά στην πύλη Azure.
